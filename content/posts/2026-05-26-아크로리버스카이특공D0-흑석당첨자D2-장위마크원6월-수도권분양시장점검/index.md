2026년 5월 26일, 수도권 분양 시장에 굵직한 이벤트가 겹쳤다. 노량진8구역 아크로 리버스카이가 오늘 특별공급 접수를 시작했고, 흑석 써밋더힐은 이틀 뒤(5/28) 당첨자 발표를 앞뒀다. 장위 푸르지오 마크원은 6월 청약을 준비 중이다. 서울 분양가가 역대 최고를 연일 경신하는 가운데, 5월 말부터 6월까지 이어지는 수도권 주요 분양 단지와 시장 흐름을 정리한다.

## 1. 5월 수도권 분양 시장 개요

부동산R114 집계 기준 5월 수도권 분양 물량은 **1만4,330가구**로 전년 동기(1만968가구) 대비 **58% 급증**했다. 지역별로는 경기 6,930가구, 인천 3,954가구, 서울 3,446가구 순이다.

물량은 늘었지만 서울 일반분양 희소성은 오히려 심화됐다.

**2026년 1-5월 서울 청약 누적 현황**

| 항목 | 수치 |
|------|------|
| 서울 일반분양 누적 물량 | 1,021가구 |
| 청약 접수 건수 | 109,110건 |
| 평균 경쟁률 | 106.87대 1 |
| 1순위 마감 단지 비율 | 10곳 중 9곳 |

서울 연간 적정 입주량(약 4만8,000가구) 대비 2026년 예정 물량은 1만8,941가구에 그친다. 구조적 공급 부족이 청약 과열의 배경이다.

## 2. 아크로 리버스카이 — 오늘(D-0) 특공 시작

노량진8구역 재개발 단지로, DL이앤씨 최상위 브랜드 '아크로'가 투입된다. 오늘 특별공급을 시작으로 이틀간 1순위 접수가 이어진다.

| 항목 | 내용 |
|------|------|
| 건설사 | DL이앤씨 |
| 총 규모 | 987가구 |
| 전용 36㎡ | 10억6,000만~11억6,370만원 |
| 전용 84㎡ | 22억6,400만~24억5,730만원 |
| 전용 140㎡ | 49억9,800만원 |
| 특공 접수 | **5월 26일(오늘)** |
| 1순위 해당지역 | 5월 27일 |
| 1순위 기타지역 | 5월 28일 |
| 당첨자 발표 | 6월 5일 |
| 정당 계약 | 6월 20일~24일 |

전용 84㎡ 분양가가 22억원대로 주담대가 원칙적으로 불가능하다. 노량진역(1·9호선) 도보권에 여의도·강남·시청 트리플 접근성을 갖추고, 국내 신규 분양 단지 중 처음으로 약 50억원대 면적(140㎡)을 포함한 점이 시장의 이목을 끌고 있다. 직전 노량진권 분양인 라클라체자이드파인이 평당 7,600만원에도 26.9대 1을 기록한 만큼 결과가 주목된다.

## 3. 흑석 써밋더힐 — 당첨자 발표 D-2

동작구 흑석11구역 재개발 단지. 흑석뉴타운의 사실상 마지막 민간 공급으로 평당 8,500만원이라는 이례적 분양가에도 실수요·투자 수요가 집중됐다.

| 항목 | 내용 |
|------|------|
| 건설사 | 대우건설 (써밋 하이엔드) |
| 총 규모 | 1,515가구 (일반분양 432가구) |
| 3.3㎡당 분양가 | 약 8,500만원 |
| 전용 59㎡ | 약 24.6억원 |
| 전용 84㎡ | 약 28.8억원 |
| 특공 | 5월 18일 완료 |
| **당첨자 발표** | **5월 28일(D-2)** |
| 입주 예정 | 2030년 6월 |
| 거주의무 | 없음 |

84㎡ 기준 총 자금 소요액은 약 27억원 이상이다. 거주의무가 없어 입주 후 전세 전환이 가능한 점이 고자산가 수요를 자극했다. 교통은 흑석역(577m)·동작역(584m) 더블역세권으로 한강변·흑석고 학군·뉴타운 희소성이 맞물린다.

## 4. 장위 푸르지오 마크원 — 6월 청약 임박

성북구 장위10구역 재개발. 강북권 1,000가구 이상 대단지 중 역대 최고 분양가로 주목받고 있다.

| 항목 | 내용 |
|------|------|
| 건설사 | 대우건설 |
| 총 규모 | 1,931가구 (일반분양 1,031가구) |
| 3.3㎡당 분양가 | 5,200~5,300만원 |
| 전용 59㎡ | 13억~14억원 |
| 전용 84㎡ | 16억5,000만~17억원 |
| 청약 예정 | 6월 |
| 교통 | 6호선 돌곶이역 도보 3분, GTX-C 예정 |

2024년 장위6구역(84㎡ 12억원) 대비 4-5억원 올랐다. 59㎡(13-14억원)는 주담대 15억 미만 구간으로 실수요 경쟁이 집중될 전망이다. 반면 84㎡(16.5-17억원)는 주담대 제한 구간으로 현금력이 당락을 가른다.

## 5. 5월 완료 단지 — e편한세상 부천 어반스퀘어

소사역(1호선·서해선 더블역세권) 인근 대단지로, 서울 분양가 절반 이하 가격에 역세권 신축을 노린 탈서울 실수요가 몰렸다.

| 타입 | 분양가 범위 | 결과 |
|------|-----------|------|
| 전용 59㎡ | 7억8,270만~8억7,080만원 | 1순위 마감 |
| 전용 74㎡ | 9억2,130만~10억1,960만원 | 1순위 마감 |
| 전용 84㎡ | 10억1,360만~11억3,900만원 | **국평 8대 1** |

본청약 경쟁률 8대 1(평균가점 55점), 사후 무순위(줍줍)는 56대 1로 마감했다. 서울 106대 1과 비교하면 수도권 내 양극화가 수치로 확인된다.

## 6. 6월 분양 예고 — 창릉 3,387가구·써밋 클라비온

| 단지 | 위치 | 물량 | 특징 |
|------|------|------|------|
| 써밋 클라비온 | 영등포구 신길10구역 | 812가구 | 7호선 신풍역 초역세권, 신길뉴타운 첫 하이엔드 |
| 고양 창릉 S-2 | 고양시 창릉신도시 | 1,057가구 | 공공분양·나눔형 |
| 고양 창릉 S-3 | 고양시 창릉신도시 | 1,306가구 | 공공분양·나눔형 |
| 고양 창릉 S-4 | 고양시 창릉신도시 | 1,024가구 | 공공분양 |

**써밋 클라비온**은 대우건설 시공의 신길뉴타운 첫 하이엔드 브랜드 공급이다. 7호선 신풍역 초역세권·전용 44-84㎡ 구성으로, 공식 분양가는 공고 전이지만 신길뉴타운 시세를 감안하면 84㎡ 기준 20억원 초중반이 예상된다.

**고양 창릉 S-2~S-4** 합산 3,387가구는 6월 공공분양 최대 공급처다. 분양가상한제·나눔형(시세 70% 이하, 5년 의무거주)으로 민간 고분양가 피로감을 흡수하는 역할이 기대된다.

## 7. 분양가 동향 — 서울 평당 5,489만원 역대 최고

원자재·인건비 급등과 신규 입주 물량 감소(2026년 1만8,941가구, 2027년 1만1,349가구)가 맞물려 서울 민간 분양가는 구조적 상승 국면이다.

| 분양가 구간 | 대표 단지 | 대출 여건 |
|------------|----------|----------|
| 평당 5,000만원대 | 장위 마크원 | 59㎡ 주담대 가능 (15억 미만) |
| 평당 7,000만원대 | 노량진 라클라체자이드파인 | 84㎡ 주담대 불가 |
| 평당 8,500만원대 | 흑석 써밋더힐, 아크로 리버스카이 | 전 타입 주담대 불가 |

15억원 기준 주담대 가능 여부가 청약 전략의 핵심 변수로 굳어졌다. 같은 단지 안에서도 59㎡(주담대 가능)와 84㎡(주담대 불가) 경쟁률 격차가 수십 배에 달하는 사례가 반복되고 있다.
