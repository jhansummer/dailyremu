2026년 5월 19일 기준, 수도권 분양 시장의 굵직한 단지들이 청약 레이스의 중반을 넘어섰다. 흑석 써밋더힐은 어제(18일) 특별공급을 마쳤고, 아크로 리버스카이(노량진)가 오는 26일 특공 시작을 일주일 앞뒀다. 장위 푸르지오 마크원은 강북 역대 최고 분양가를 들고 시장의 반응을 확인 중이다. 5월 하반기와 6월 분양 일정을 단지별로 총정리한다.

## 1. 5월 분양 시장 현황 — 서울 평균 경쟁률 106대 1

부동산R114 집계 기준 5월 수도권 분양 물량은 **1만4,330가구**로 전년 동기 대비 58% 급증했다. 물량은 늘었지만 서울 일반분양의 희소성은 오히려 심화됐다.

**2026년 1-5월 서울 청약 현황**

| 항목 | 수치 |
|------|------|
| 서울 일반분양 누적 물량 | 1,021가구 |
| 청약 접수 건수 | 109,110건 |
| 평균 경쟁률 | 106.87대 1 |
| 1순위 마감 단지 비율 | 10곳 중 9곳 |

연간 적정 입주량(약 4만8,000가구) 대비 2026년 서울 예정 물량은 1만8,941가구에 그친다. 공급 구조적 부족 속에서 신규 분양 희소성이 청약 과열을 떠받치는 구조다. 평균 분양가는 3.3㎡당 **5,489만원**으로 역대 최고치가 이어지고 있다.

## 2. 흑석 써밋더힐 — 특공 완료, 당첨자 발표 D-9

동작구 흑석11구역 재개발 단지로, 흑석뉴타운의 사실상 마지막 민간 공급 물량이다.

| 항목 | 내용 |
|------|------|
| 건설사 | 대우건설 (써밋 하이엔드 브랜드) |
| 총 규모 | 1,515가구 (일반분양 432가구) |
| 3.3㎡당 분양가 | 약 8,500만원 |
| 전용 59㎡ | 약 24.6억원 |
| 전용 84㎡ | 약 28.8억원 |
| 특공 | 5월 18일 완료 |
| 당첨자 발표 | 5월 28일 |
| 입주 예정 | 2030년 6월 |
| 거주의무 | 없음 |
| 교통 | 흑석역(577m), 동작역(584m) |

84㎡ 기준 총 자금 소요액은 약 27억원 이상이다. 거주의무가 없어 입주 후 전세 전환이 가능한 점이 고자산가 수요를 자극한다. 직전 흑석권역 분양 단지인 노량진뉴타운 라클라체자이드파인이 평당 7,600만원에도 **26.9대 1**의 경쟁률을 기록했던 만큼, 한강변·학군(흑석고)·뉴타운 마지막 희소성 3박자가 맞아떨어지는 단지다.

## 3. 아크로 리버스카이 — 특공 D-7, 50억짜리 펜트하우스도

노량진8구역 재개발 단지로, DL이앤씨 최상위 브랜드 '아크로'가 투입된다.

| 항목 | 내용 |
|------|------|
| 건설사 | DL이앤씨 |
| 총 규모 | 987가구 |
| 전용 36㎡ | 10억6,000만~11억6,370만원 |
| 전용 84㎡ | 22억6,400만~24억5,730만원 |
| 전용 140㎡ | 49억9,800만원 |
| 특공 접수 | 5월 26일 |
| 1순위 해당지역 | 5월 27일 |
| 1순위 기타지역 | 5월 28일 |
| 당첨자 발표 | 6월 5일 |
| 정당 계약 | 6월 20일~24일 |

노량진역(1·9호선) 도보권에 여의도·강남·시청 트리플 접근성을 갖춘다. 전용 84㎡가 20억원 중반대로 주담대가 원칙적으로 불가능하지만, 올해 서울 경쟁률 추이를 감안하면 고가점·고자산가 수요가 집중될 것으로 예상된다. 국내 최초로 50억에 근접한 신규 분양 물량(140㎡)도 포함돼 시장의 이목을 끌고 있다.

## 4. 장위 푸르지오 마크원 — 강북 최고 분양가 1,031가구

성북구 장위10구역 재개발. 강북권 1,000가구 이상 대단지 기준 역대 최고 분양가를 예고한 단지다.

| 항목 | 내용 |
|------|------|
| 건설사 | 대우건설 |
| 총 규모 | 1,931가구 (일반분양 1,031가구) |
| 3.3㎡당 분양가 | 5,200~5,300만원 |
| 전용 59㎡ | 13억~14억원 |
| 전용 84㎡ | 16억5,000만~17억원 |
| 교통 | 6호선 돌곶이역 도보 3분, GTX-C 예정 |

2024년 같은 장위 6구역 분양가(12억원)와 비교하면 84㎡ 기준 4-5억원 올랐다. 현지 중개업계는 "예상보다 2-3억 이상 높다"며 신중론을 내놨다. 59㎡(13-14억)는 주담대 적용이 가능한 15억원 미만 구간이라 실수요 경쟁이 집중되는 반면, 84㎡는 주담대 제한(15억 초과)으로 자금 동원력이 당락을 가른다. GTX-C 예정역이라는 미래 가치와 고분양가 현재 부담 사이에서 옥석가리기가 진행 중이다.

## 5. e편한세상 부천 어반스퀘어 — 국평 8대 1 마감

소사역(1호선·서해선 더블역세권) 인근 대단지로, 서울 분양가의 절반 이하 가격에 역세권 신축을 노릴 수 있어 탈서울 실수요가 몰린 단지다. 5월 6일 1순위가 마감됐고 13일 당첨자가 발표됐다.

| 타입 | 분양가 범위 | 결과 |
|------|-----------|------|
| 전용 59㎡ | 7억8,270만~8억7,080만원 | 1순위 마감 |
| 전용 74㎡ | 9억2,130만~10억1,960만원 | 1순위 마감 |
| 전용 84㎡ | 10억1,360만~11억3,900만원 | 국평 경쟁률 8대 1 |

- **1순위 국평(84㎡) 평균 경쟁률**: 약 8대 1, 평균가점 55점
- **사후 무순위(줍줍)**: 경쟁률 56대 1로 마감

서울 84㎡ 절반 이하 가격에 GTX-B 부천종합운동장역 개통 기대감까지 더해져 안정적 청약 결과를 기록했다. 단, 본청약 경쟁률(8대 1)은 서울 평균(106대 1) 대비 낮아, 서울과 경기 분양 시장의 양극화가 수치로 확인된 사례이기도 하다.

## 6. 6월 분양 미리보기 — 신길 써밋 클라비온 + 창릉 3,387가구

| 단지 | 위치 | 물량 | 특징 |
|------|------|------|------|
| 써밋 클라비온 | 영등포구 신길10구역 | 812가구 | 7호선 신풍역 초역세권, 신길뉴타운 첫 하이엔드 |
| 고양 창릉 S-2 | 고양시 창릉신도시 | 1,057가구 | 공공분양·나눔형 |
| 고양 창릉 S-3 | 고양시 창릉신도시 | 1,306가구 | 공공분양·나눔형 |
| 고양 창릉 S-4 | 고양시 창릉신도시 | 1,024가구 | 공공분양 |
| 부천 역곡 | 부천시 | 미정 | 공공분양 |

**써밋 클라비온**은 대우건설이 시공하는 신길뉴타운 첫 하이엔드 공급 단지다. 7호선 신풍역 초역세권에 총 812가구(전용 44-84㎡)로, 아직 공식 분양가는 공고 전이나 신길뉴타운 시세를 감안하면 84㎡ 기준 20억원 초중반이 예상된다.

**고양 창릉 S-2~S-4** 합산 3,387가구는 6월 공공분양의 최대 공급처다. GTX-A 고양은평선 등 광역 교통망 수혜가 예정되어 있으며, 나눔형은 시세 70% 이하 분양가에 5년 의무거주 조건이 적용된다.

## 7. 분양가 동향 — 서울 고분양가 구조화

원자재·인건비 급등과 신규 입주 물량 감소(2026년 1만8,941가구, 2027년 1만1,349가구)가 맞물려 서울 민간 분양가는 구조적 상승 국면에 접어들었다.

| 구간 | 대표 단지 | 대출 여건 |
|------|----------|----------|
| 평당 5,000만원대 | 장위 마크원, 흑석(강북) | 15억 미만 타입 주담대 가능 |
| 평당 7,000만원대 | 흑석 써밋더힐 | 주담대 불가 구간 중심 |
| 평당 8,000만원대 이상 | 아크로 리버스카이, 흑석 | 고자산가 현금 중심 |

분양가상한제 적용 공공분양(창릉·왕숙 등)이 민간 분양가 상승세를 일부 흡수하는 구도가 이어지고 있다. 수요자로서는 본인 자금 여력과 자격 요건(특공 유무, 청약가점)을 기준으로 민간·공공 분양을 병행 검토하는 전략이 현실적이다.
